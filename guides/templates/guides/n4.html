Guide #3 - Loops: "For" and "While" statements.

Ok. A última lição foi sobre estruturas condicionais. Agora, veremos estruturas de repetição ( loops ). São utilizadas quando existe a necessidade de se executar um mesmo bloco de código diversas vezes. Para isso, usaremos os comandos for e while : 
int i;
for (i = 0; i < 10; i++) {
  //Código
}

while (c) {
  //Código
}

O que é isso ?

Vamos analisar o primeiro caso. O for , antes de tudo, necessita de uma variável, para controlar o número de repetições desejado. Na maioria das vezes, esta variável será um número inteiro, mas também pode ser outros tipos (como char , por exemplo). Note que dentro dos parênteses do for , há 3 "argumentos". O primeiro é a atribuição inicial da sua variável: qual valor ela terá ao começar este laço de repetição (e está definição ocorre apenas uma vez). O segundo é a condição de continuidade do laço, ou seja, enquanto esta condição for verdadeira, o for continuará executando. Por último (e opcional), é a alteração do estado da sua variável. Neste caso, a variável é incrementada sempre que se chega ao fim do laço. Aliás, i++ é o mesmo que i = i + 1 (ou seja, adicionando 1 ao valor da variável). Portanto, o corpo deste for será executado 10 vezes: 
int i;
for (i = 0; i < 10; i++) {
  //Primeiramente, o valor de i será 0.
  //Ao chegar no fim do laço, o valor de i é incrementado (ou seja, i = 1)
  //Como a condição (i < 10) é verdadeira, o laço é executado novamente.
  //... E, novamente, ao fim do laço, o valor de i é incrementado e passa a ser 2.
  //Ou seja, quando o valor de i alcançar 10, a condição (10 < 10) será falsa.
  //Quando esta condição for falsa, o laço de repetição termina.
}
//Resto do código
A variável i é uma variável igual qualquer outra, e podemos utilizá-la dentro do laço: 
int i;
for (i = 0; i < 10; i++) {
  printf("O valor de i é: %i\n", i);
}
//Resto do código
Podemos alterar a variação de i da maneira que desejarmos: 
int i;
for (i = 0; i < 10; i+=2) {
  //Neste caso, é adicionado 2 ao valor de i a cada fim de iteração (repetição).
  //Logo, neste caso, o laço será executado apenas 5 vezes.
}

for (i = 10; i > 0; i--) {
  //Mesmo esquema, porém i começa com o valor em 10, e vai decrementando até 0.
  //Executará 10 vezes.
  //i-- é o mesmo que i = i - 1
}
//Resto do código

Entendi. E o while ?

O while funciona de maneira semelhante ao for , sendo utilizado para criar estruturas de repetição. Mas sua sintaxe é mais simples, lembrando uma estrutura condicional ( if ): 
while (c) {
  //Código            
}
O corpo do while será executado sempre que a condição c for verdadeira. Com isso, podemos implementar uma estrutura que possui um comportamento equivalente ao de um for , porém, utilizando um while : 
int C = 0;
while (C < 10) {
  //Código  
  C++;
}

for (C = 0; C < 10; C++) {
  //Código
}
Ambos os laços terão o mesmo comportamento.

Então qual a diferença ? Qual dos dois devo usar ?

Ambos são utilizados para repetir a execução de um pedaço do código. Use o for em situações em que você precise "contar" alguma variável, ou controlar os passos de sua variável de maneira padronizada: por exemplo, se eu utilizar for (i = 0; i <= 20; i+=2) , terei todos os valores pares, de 0 até 20 (note que, neste caso, estamos incluindo o 20!). 
int N, i;
scanf("%i", &N);
for (i = 0; i < N; i++) {
  //Código
  //Neste caso, estamos iterando até um determinado valor definido na entrada.
}
Use o while quando a variação da sua variável de controle não segue uma "contagem" ou algum tipo de padrão.

Ainda existem outros dois comandos para auxiliar no controle de seus laços de repetição: break e continue . 
int i;
for (i = 0; i < 10; i++) {
  if (i == 5) {
    break;
  }
}
//Resto do código
O break serve para interromper o laço prematuramente. No exemplo acima, se o valor de i for 5, o condicional será verdadeiro e consequentemente será executado. Ao cair no break o código imediatamente sai do laço (e continua na linha 7), mesmo que a condição (5 < 10) seja verdadeira. Portanto, o laço se repetirá 5 vezes (e não 10).
O continue serve para ignorar determinados casos no laço: 
int i;
for (i = 0; i < 10; i++) {
  if (i == 5) {
    continue;
  }

  printf("%i\n", i);
}
No exemplo acima, o laço irá printar na tela 0, 1, 2, 3, 4, 6, 7, 8, 9. Quando o valor de i for 5, o condicional será verdadeiro e executará o continue . Ao invés de terminar o laço (como aconteceria no break ), o comando força o laço a ir para sua próxima iteração, ignorando o restante do corpo do for (No exemplo acima, o printf não será executado se o valor de i for 5). Neste caso, o laço se repetirá 10 vezes normalmente (mesmo que apareça apenas 9 números).
